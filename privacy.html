Setsuna Privacy Policy

Setsuna is a desktop application for reading Japanese text.

Data Collection

Setsuna does not collect or store personal user data.

Local Storage

All user data (settings, dictionaries, etc.) is stored locally on the user's device.

Google Drive Integration

If enabled, Setsuna may store application data in the user's Google Drive AppData folder. This data is private and not accessible to others.

Third Parties

No user data is shared with third parties.

Contact

If you have questions, contact the developer via GitHub.
