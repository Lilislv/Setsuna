Setsuna — десктоп-приложение для чтения японского текста.

## Функции

- Лукан слов как в Yomitan
- Импорт словарей Yomitan
- Интеграция с Anki
- Создание карточек в 1 клик
- Статистика чтения
- Встроенный браузер
- Синхронизация через Google Drive AppData
