# Setsuna

[English](README.md)

---

Setsuna — десктоп приложение для чтения японского текста.

## Функции

- Лукап слов (как Yomitan)
- Импорт словарей Yomitan
- Интеграция с Anki (карточки в 1 клик)
- Статистика чтения
- Встроенный браузер

## Скачать

➡️ Releases: https://github.com/Lilislv/Setsuna/releases

## Политика конфиденциальности

Setsuna хранит данные локально и опционально в Google Drive (AppData).

Личные данные не собираются и не передаются третьим лицам.

## Условия использования

Программа предоставляется "как есть", без гарантий.
