Privacy Policy

Last updated: April 30, 2026

Overview

Setsuna is a desktop application for reading Japanese text, looking up words, tracking reading progress, and creating Anki cards.

This policy explains what data the app handles and how optional Google Drive synchronization works.

Data Collection

Setsuna does not operate a server that collects, sells, or shares personal user data.

The app stores settings, reading data, dictionary metadata, and card-mining information locally on the user's device.

Google Drive Integration

If the user enables cloud sync, Setsuna may store application data in the user's Google Drive AppData folder.

This data is used only to synchronize Setsuna settings and app data between the user's own devices. Setsuna does not request access to the user's general Drive files.

Third Parties

Setsuna may connect to local tools selected by the user, such as Anki or text-hooking software. Data sent to those tools is controlled by the user and their local setup.

No user data is sold to third parties.

Contact

For privacy questions, contact the developer through GitHub or by email at setsunalookup@gmail.com.
