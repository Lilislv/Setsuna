[Русский](README_RU.md)

---

Setsuna is a desktop app for reading Japanese text.

## Features

- Word lookup (Yomitan-like)
- Yomitan dictionary import
- Anki integration (1-click mining)
- Reading statistics
- Built-in browser
- Google Drive AppData sync
