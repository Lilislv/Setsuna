# Setsuna

<details open>
<summary>🇬🇧 English</summary>

---

Setsuna is a desktop app for reading Japanese text.

## Features

- Word lookup (Yomitan-like)
- Yomitan dictionary import
- Anki integration (1-click mining)
- Reading statistics
- Built-in browser

## Download

➡️ https://github.com/Lilislv/Setsuna/releases
